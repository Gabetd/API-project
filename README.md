
# DayBnB

 DayBnB is a clone of the AirBnB app.

## Features
* Spots:
create Spot
show Spot
update Spot
delete Spot
*reviews
create Review
show Review
update Review
delete Review
*Bookings
create Booking
show Booking
update Booking
delete Booking
